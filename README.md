# signature-builder

Company email signature "builder".


### Available NPM tasks
Start the project in development mode (live reload)
```bash
$ yarn start
```

Create a static build in the `./dist` folder
```bash
$ yarn build
```

<!-- Run the tests
```bash
$ yarn test
``` -->

Run the build action and deploy to GitHub pages 
```bash
$ yarn deploy
```
